About the Project

    This project analyzes chat data (such as WhatsApp or Telegram exports) to find and visualize the most frequently used emojis. It demonstrates how text data can be cleaned, processed, and turned into meaningful insights using Python.

Key Features

    Reads chat files and extracts only the messages.

    Counts how often each emoji appears using the emoji Python library and regex.

    Identifies the top 5 most used emojis in the conversation.

    Creates visualizations (bar and pie charts) with Matplotlib.

    Helps understand emoji usage trends in group chats.

Tech Stack

    Python
    Regular Expressions (re)
    Emoji library
    Matplotlib

Outcome :

    The project gives a fun yet practical example of text analysis and data visualization. It highlights how emojis reflect the tone and style of digital conversations.
